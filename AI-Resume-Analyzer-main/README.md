## Setup & Installation ğŸ‘€

### Step 1: Clone or Download the Project
```bash
git clone https://github.com/your-repo/AI-Resume-Analyzer.git
cd AI-Resume-Analyzer
```

### Step 2: Create Virtual Environment
```bash
python -m venv venv
```

### Step 3: Activate Virtual Environment

**Windows:**
```bash
venv\Scripts\activate
```

**Linux/Mac:**
```bash
source venv/bin/activate
```

### Step 4: Install Dependencies
```bash
cd App
pip install -r requirements.txt
python -m spacy download en_core_web_sm
```

### Step 5: Apply PyResParser Fix
Copy the custom `resume_parser.py` to the installed package:
```bash
copy ..\pyresparser\resume_parser.py ..\venv\Lib\site-packages\pyresparser\resume_parser.py
```

**Note:** The project automatically creates the SQLite database (`cv.db`) on first run.

### Step 6: Run the Application
```bash
streamlit run App.py
```

The application will open in your default browser at `http://localhost:8501`

## Application Flow ğŸ”„

### 1. **User Section** ğŸ‘¤
- Enter your name, email, and mobile number
- Upload your resume in PDF format
- The system will:
  - Parse and extract information (name, email, skills, education, experience)
  - Predict your experience level (Fresher/Intermediate/Experienced)
  - Recommend skills based on your field (Data Science, Web Dev, Android, iOS, UI/UX)
  - Provide course recommendations
  - Calculate resume score based on sections present
  - Show resume writing tips and interview preparation videos

### 2. **Feedback Section** ğŸ’¬
- Submit feedback about the tool
- Rate the application (1-5 stars)
- View past user ratings and comments

### 3. **About Section** â„¹ï¸
- Learn about the tool's purpose and features
- View usage instructions

### 4. **Admin Section** ğŸ”

#### First Time Setup:
1. Select **"Create Admin"** from the dropdown
2. Enter a new username and password
3. Click **"Create"** to register

#### Admin Login:
1. Select **"Login"** from the dropdown
2. Enter your credentials
3. Access the admin dashboard with:
   - User analytics and data tables
   - Resume score distribution
   - Field predictions (pie charts)
   - Geographic usage statistics
   - Feedback analysis
   - CSV export functionality
